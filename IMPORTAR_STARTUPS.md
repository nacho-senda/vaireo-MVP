# IMPORTAR 100+ STARTUPS - INSTRUCCIONES RÁPIDAS

## MÉTODO MÁS FÁCIL

1. **Abre tu navegador**
2. **Ve a:** `http://localhost:3000/admin/import-startups`
3. **Haz clic en:** "Importar Todo" (botón grande)
4. **Espera 2-3 minutos**
5. **Listo!** Tendrás 100+ startups en tu base de datos

## VERIFICAR

Después de importar, visita:
- `/startups` - Ver todas las startups
- `/analytics` - Ver estadísticas

## ALTERNATIVA: COMANDO

```bash
node scripts/populate-db.mjs
```
