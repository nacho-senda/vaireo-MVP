# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2025-01-XX

### Añadido
- Plataforma inicial del ecosistema agroalimentario español
- Directorio de 75 startups del sector AgriFood/FoodTech
- Sistema de filtros avanzados (vertical, región, tecnología, ODS)
- Página de analytics con visualizaciones y métricas del ecosistema
- Sistema de gestión de proyectos colaborativos
- Asistente IA con acceso a base de datos de startups
- Integración con Google Sheets para carga dinámica de datos
- Conector HubSpot CRM (draft para futuras integraciones)
- Modo oscuro/claro con persistencia
- Diseño responsive y accesible
- Perfiles detallados de startups con información completa
- Sistema de registro de usuarios conectado a Google Sheets

### Características Técnicas
- Next.js 14 con App Router
- React 19 y TypeScript
- Tailwind CSS v4 para estilos
- shadcn/ui para componentes UI
- Vercel AI SDK v5 para el asistente
- Recharts para visualización de datos
- Server Actions para operaciones del servidor
- Optimización de imágenes y fuentes

### Documentación
- Guía completa de instalación y configuración
- Documentación de integración con Google Sheets
- Guía de configuración HubSpot
